Beispiel "Hello World"

In diesem Beispiel werden mehrere Versionen eines in C# geschriebenen Programms mit der Bezeichnung "Hello World" vorgestellt.

Sicherheitshinweis

Dieser Beispielcode dient dazu, ein Konzept zu veranschaulichen. Er sollte nicht für Anwendungen oder Websites verwendet werden, da er unter Umständen nicht den sichersten Programmiermethoden entspricht. Microsoft übernimmt keine Haftung für beiläufig entstandene Schäden oder Folgeschäden, falls der Beispielcode nicht bestimmungsgemäß verwendet wird.

So erstellen Sie die "Hello World"-Beispiele in Visual Studio und führen sie aus

  1. Öffnen Sie die Projektmappe (HelloWorld.sln).

  2. Klicken Sie im Projektmappen-Explorer mit der rechten Maustaste auf das Projekt HelloWorld1, und wählen Sie Als Startprojekt festlegen aus.

  3. Klicken Sie im Menü Debuggen auf Starten ohne Debugging.

  4. Drücken Sie eine beliebige Taste, um HelloWorld1 zu schließen.

  5. Klicken Sie im Projektmappen-Explorer mit der rechten Maustaste auf das Projekt HelloWorld2, und wählen Sie Als Startprojekt festlegen aus.

  6. Klicken Sie im Menü Debuggen auf Starten ohne Debuggen.

  7. Drücken Sie eine beliebige Taste, um HelloWorld2 zu schließen.

  8. Klicken Sie im Projektmappen-Explorer mit der rechten Maustaste auf das Projekt HelloWorld3, und wählen Sie Als Startprojekt festlegen aus.

  9. Klicken Sie im Projektmappen-Explorer mit der rechten Maustaste auf das Projekt HelloWorld3, und klicken Sie auf Eigenschaften.

  10. Öffnen Sie den Ordner Konfigurationseigenschaften, und klicken Sie auf Debuggen.

  11. Geben Sie "A B C" in der Eigenschaft "Befehlszeilenargumente" ein, und klicken Sie auf OK.

  12. Klicken Sie im Menü Debuggen auf Starten ohne Debuggen.

  13. Drücken Sie eine beliebige Taste, um HelloWorld3 zu schließen.

  14. Klicken Sie im Projektmappen-Explorer mit der rechten Maustaste auf das Projekt HelloWorld4, und wählen Sie Als Startprojekt festlegen aus.

  15. Klicken Sie im Menü Debuggen auf Starten ohne Debuggen.

  16. Drücken Sie eine beliebige Taste, um HelloWorld4 zu schließen.

So erstellen Sie die "Hello World"-Beispiele über die Befehlszeile und führen sie aus

  1. Wechseln Sie mithilfe des Befehls Change Directory in das Verzeichnis HelloWorld.

  2. Geben Sie Folgendes ein: 

    cd HelloWorld1
csc Hello1.cs
Hello1

  3. Geben Sie Folgendes ein: 

    cd ..\HelloWorld2
csc Hello2.cs
Hello2

  4. Geben Sie Folgendes ein: 

    cd ..\HelloWorld3
csc Hello3.cs
Hello3 A B C D

  5. Geben Sie Folgendes ein: 

    cd ..\HelloWorld4
csc Hello4.cs
Hello4
